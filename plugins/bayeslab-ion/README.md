# Bayeslab ION Plugin

This Codex plugin combines the production Bayeslab ION remote MCP server with
a focused research workflow.

The server exposes published report lookup, period-aware comparison, industry
methodology, and isolated analysis jobs. OAuth binds access to the signed-in
GoodZ account. Creating an analysis job consumes usage credits; reading or
polling should not create a second job.

See https://bayeslab.xyz/agents/codex for setup and product details.
